# Hostel Complaint Management System

A full-stack web application with separate frontend and backend.

## Project Structure
- frontend/ – User interface
- backend/ – API & database logic

## Tech Stack
- Frontend: HTML/CSS/JavaScript (or React)
- Backend: Node.js, Express
- Database: PostgreSQL

## Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env
node server.js
